# Commerzi

Le projet "Commerzi" est une application mobile de gestion des tournées pour les commerciaux, conçue pour aider les représentants à optimiser leurs trajets, à minimiser les distances parcourues et à améliorer le démarchage de nouveaux clients (prospects). Voici une description générale du projet, de ses fonctionnalités et de ses exigences en suivant la structure fournie dans le cahier des charges.
